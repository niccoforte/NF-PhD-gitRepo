1) 2D Geometry : all type of lattice generation (FCC, Triangular, Kagome and Hexagonal) in 2D for triaxiality study.

2) 3D Geometry : FCC lattice in 3D, only nodal coordinates and element matrix generation, then can be used to copy/paste in INP file of abaqus to model.

3) TensilePython : python codes to generate 2D - FCC and Triangular lattices under tension with damage material models

4) FractureToughness : python codes to generate 2D - FCC, Triangular and Kagome lattices with edge crack. Note that the CT specimen size will standerize as per unit cells used in x and y direction. TIP: use odd numbers in y direction

5) ANN_Tensile_coordDis : MATLAB codes for neural networks optimization with input and output data. This folder contains optimization for spatial coordinates under tensile load. 

6) ANN_Tensile_sizePara : same as above but for size variable

7) ANN_FractureToughness : optimization codes for fractue toughness, material is modelled is brittle and has results for FCC, triangular and kagome lattices

Note: all ANN MATLAB codes sould be running in this order : 1-DataProcessing, 2-DeepLearning_stress_strain/DeepLearning_FractureToughness, 3-DeepLearningOptimization

After optimization you have to run python/matlab codes to generate optimized geometry.
